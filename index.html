What is Python?

Python is a high-level pragramming language which is open soutrce and object oriented. Python was developed by Guido Van Rossum in 1991. It is presently owned by Python Software Foundation(PSF).

Modes of Python

Script Mode


In script mode, You write your code in a text file then save it with a . py extension which stands for "Python" If you are in the standard Python shell, you can click "File" then choose "New" or simply hit "Ctrl + N" on your keyboard to open a blank script in which you can write your code.

Immediate Mode

When commands are entered directly in IDLE from the keyboard, the interpreter/IDLE is said to be in interactive mode. In this mode, thype the :Hello" in front of the prompt(>>>) and press Enter. This will print "Hello" on the desktop.

Variables

Variables in Python


Variable are named memory locations whose value may change during the execution of the program. Some rules for naming variables in Python are given below.

  • A variable name can consist of alphabetrs, digits and underscore.(_)
  • A variable name can start with an alphabet/ underscore(_) but not with a digit.
  • A variable name cannot have space in it.
  • Keywords are not allowed as a variable name.
  • Python is a case sensitive language, so variables in uppercase are different to those in llowercase.

Declaring Variables


Every Variable has a type and this type defines the format and behaviour of the variable.
For Example, if we write x=3.
It means value 3 will be assigned to the integer variable x automatically.
Similarly, if we write x=15 and then write >>>x and press the Enter key, value 15 will be displayed as output in the PYthon Shell.
